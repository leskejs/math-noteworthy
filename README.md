# Mathematics II

A mathematics textbook project built using the [sihooleebd/noteworthy](https://github.com/sihooleebd/noteworthy) framework.

## Build

[Typst](https://typst.app/) and [poppler](https://poppler.freedesktop.org/) need to be installed.
On windows, curses (which is used by `noteworthy.py`) can be installed with `pip install windows-curses`.

If all dependencies are met, simply run
```bash
python3 noteworthy.py
```
